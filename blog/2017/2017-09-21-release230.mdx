:::caution

This release contains an updated internal framework file.

:::

- **Android Oreo Support** ([#1594](https://github.com/iBotPeaches/Apktool/issues/1594))
- Run CI tests on OSX as well as unix. ([#1571](https://github.com/iBotPeaches/Apktool/issues/1571))
- Prevent apktool doing permission checks on unneeded directories. ([#1586](https://github.com/iBotPeaches/Apktool/issues/1586))
- Windows Unicode support for helper scripts. ([#1595](https://github.com/iBotPeaches/Apktool/issues/1595))
- Dropped support for all 32 bit binaries, except Windows ([#1598](https://github.com/iBotPeaches/Apktool/issues/1598))
- Fixed `ClassCastExceptions` when producing files. ([#1444](https://github.com/iBotPeaches/Apktool/issues/1444)) / Thanks xpirt
- Fixed unclosed streams when using `SmaliMod`. Thanks footpatch
- Cleanup deprecations in Apache Commons CLI. Thanks xpirt
- Fixed issue with apks that have 4 non-positional/positional nodes. ([#1611](https://github.com/iBotPeaches/Apktool/issues/1611))
- Added support for invalid `targetSdkValues` which are greater than `max` or less than `min`. ([#1615](https://github.com/iBotPeaches/Apktool/pull/1615)) / Thanks Jan Vidar Krey
- Added ability to skip assets decode (`--no-assets`) ([#1605](https://github.com/iBotPeaches/Apktool/issues/1605))
- Prevented directories being written outside of scope of decode directory. ([#1589](https://github.com/iBotPeaches/Apktool/issues/1589))
